## 프로젝트 소개

---

낯선 곳을 여행할 때마다 맛집, 액티비티, 주변 명소를 찾느라 한참을 검색하곤 했어요.<br>
그런데 문득 이런 생각이 들었죠.<br>
'만약 이 도시를 잘 아는 친구가 내 코스를 계획해준다면?'<br><br>
이 작은 호기심에서 우리의 이야기가 시작됐어요.
